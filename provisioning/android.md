# Android Apps

Recommended apps for using svalbard drives on Android.

## Essential

- **Kiwix** — Browse ZIM encyclopedias and references offline
- **OsmAnd** — Offline maps (can import PMTiles)

## Reading

- **ReadEra** — EPUB/PDF reader
- **Moon+ Reader** — Feature-rich ebook reader

## AI

- **ChatterUI** — Run GGUF models locally on-device
